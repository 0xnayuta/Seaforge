# 项目定位

---

## 项目名称

纵横四海 (Seaforge)

## 项目类型

单人离线航海贸易经营 × RPG。

贸易是经济轴，RPG 是成长轴。玩家通过跑商积累财富，用于购买装备、招募 NPC、挑战副本；通过战斗/任务/探索获取经验与装备，反过来解锁更高收益的贸易路线和更有挑战的内容。

## 灵感来源

- QQ 家园《纵横四海》——借鉴「航海贸易经营」核心循环，不复制原内容
- 《艾尔登法环》——属性 Build 自由度和装备收集驱动力
- 《BLACK SOULS》——回合制策略战斗设计、黑暗童话风格取向

约束：
- 仅借鉴核心玩法与设计理念
- 不得复制原游戏文案、美术、角色、剧情、地图设计
- 不追求与原版一致的体验，只借鉴其核心循环和成长框架

## 核心循环（双轴）

```
经济轴：                        成长轴：
  查看港口价格                     接取任务 / 进入副本
    ↓                                ↓
  购买商品                         回合制战斗（人物）
    ↓                                ↓
  选择目的港                       获得经验 / 装备 / 物品
    ↓                                ↓
  航行（舰队战/接舷战/随机事件）      提升人物属性 / 替换装备
    ↓                                ↓
  抵达港口 → 出售商品 → 获得利润      挑战更难的内容
    ↓
  购买装备 / 补给 → 继续贸易

交汇点：贸易赚钱 → 支撑装备消费 / 人物成长 → 解锁高收益贸易路线
```

## 非目标

- 不是复刻版 — 借鉴玩法，不复制原内容
- 不是即时反应游戏 — 回合制/文字决策是核心体验
- 不是社交游戏 — 单人离线项目
- 不是 MMORPG — 单人离线项目
- 不是动作游戏 — 所有战斗为回合制/半即时制，无实时操作判定，无帧率相关精确输入（回避/弹反是技能概率，不是玩家反应时间）
- 不是生存游戏 — 无饥渴/体温/疾病/补给管理系统
- 不是基地建设 — 无城建/生产链/自动化系统
- 非无缝开放世界 — 世界由离散地点（港口、副本、地标）组成，通过菜单/航海图导航；但单个地点内部可以有探索空间
- 无实时等待 — 所有操作即时生效，航行瞬间结算

## 目标平台

PC

## 目标用户画像

- 曾在 QQ 家园玩过纵横四海的老玩家，想重温跑商体验
- 喜欢大航海时代系列、海商王等贸易模拟游戏的玩家
- 喜欢魂系/RPG 属性配装、Build 多样性的玩家
- 喜欢回合制策略战斗（非实时操作）的玩家
- 喜欢收集装备、探索地图、挑战副本的玩家
- 喜欢文字界面 + 策略决策的玩家
- 不需要社交、只想安静玩一局的玩家
